## Usage

### Step 1
Add it in your root build.gradle at the end of repositories:
```
allprojects {
  repositories {
    ...
    maven { url 'https://jitpack.io' }
  }
}
``` 
Add dependency in your app level build.gradle.
``` 
dependencies {
    implementation 'com.github.andreasagap:LoadingButton:1.0'
}
```
